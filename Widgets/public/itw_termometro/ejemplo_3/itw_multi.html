Multiples widgets


Este ejemplo usa JQuery para redimensionar y desplazar los widgets.

Por defecto, se setea el widget en position absolute. Esto se debe a que, por como funciona JQuery, si se desplaza el widget, los demas objetos no se enteran del cambio, siguen comportandose como si siguiera habiendo un widget en el lugar original. Luego, al redimensionarlo, los otros elementos de la pagina se acomodan de una manera poco intuitiva para el usuario, "creen" ser empujados. En otras palabras, al mover el widget con el mouse usando JQuery, el widget se dibuja en un plano por encima del resto de los elementos. Pero, para que los demas componentes del sitio se enteren de ello, es necesario "avisarles" colocando position absolute al widget.

Para los input, se crean eventos usando el metodo forEach. Prestar atension al alcance de las variables, es recomendable usar ForEach() en vez de for.
